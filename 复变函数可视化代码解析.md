
复变函数$f:\mathbb{C} \to \mathbb{C}$ 是复数到复数的映射，定义域需要二维表示，而值域也需要二维表示。像二维实函数一样，复变函数的定义域常用平面坐标$(x,y)$来描述，而值域在动画表示中也用平面坐标描述（双图），在静态图中常用色相(Hue) 表示$r$, 饱和度(Saturation) 来表示$\theta$。
$$
x + y \mathrm{i} \mapsto r \mathrm{e}^{\mathrm{i} \theta}
$$


## Gnuplot 代码
```
############### 默认设置画面 #######################
set term svg font 'Arial,16'

set encoding utf8
set xlabel 'Re($z$)' offset 0, 0.5
set ylabel 'Im($z$)' offset 0.5,0
unset key
set view map scale 1
set isosamples 150, 150
set samples 150
set size square 0.9
set style data lines
set cblabel "Phase angle"
set cblabel  offset character -2, 0, 0 font "" textcolor lt -1 rotate by -270
set cbrange [ -pi : pi ] noreverse nowriteback
set cbtics   ('-$\\pi$' -pi, '$\\pi$' pi)
set palette defined ( 0 0 1 1, 1 1 1 1 )
set palette positive nops_allcF maxcolors 0 gamma 1.5 color model HSV
############### 默认设置画面 #######################


set xrange [-6:6] noreverse nowriteback
set yrange [-6:6] noreverse nowriteback

Hue(x,y) = (pi + atan2(y,x)) / (2*pi)
phase(x,y) = hsv2rgb( Hue(x,y), sqrt(x**2+y**2) )
a = 2
b = 3
alpha = 0.5
t = 1.5
phi = pi / 4
n = 1
f(x,y) = exp((x + y*{0,1}) * ({0,1})) / ( 1 + (x + y*{0,1})**2 )**3
rp(x,y) = real(f(x,y))
ip(x,y) = imag(f(x,y))
color(x,y) = hsv2rgb( Hue( rp(x,y), ip(x,y) ), abs(f(x,y)), 1)
#color(x,y) = hsv2rgb( Hue( rp(x,y), ip(x,y) ), 1, abs( f(x,y) ) )
set urange [-6:6]
set vrange [-6:6]
set xrange [-6:6]
set yrange [-6:6]
set tics rangelimit scale 0.0
set xtics 3 offset 0, 0.5
set ytics 3 offset 0.5, 0
set arrow from -4,0 to 4,0 nohead front
#set arrow from -9,2*pi to 9,2*pi nohead front
set out "cosx1x23.svg"
splot '++' using 1:2:(color($1,$2)) with pm3d lc rgb variable
```
